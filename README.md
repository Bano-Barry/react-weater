# Application Météo React

## Vue d'ensemble du projet

Cette application web est une application météo simple développée avec React et Vite. Elle permet d'afficher les informations météorologiques actuelles pour la ville la plus proche de l'utilisateur, en utilisant l'API AirVisual. L'application est conçue pour être responsive et affiche la température, la ville, le pays et une icône représentative de la météo.

### Fonctionnalités principales

- **Affichage des données météo** : Récupération automatique des données météorologiques pour la ville la plus proche.
- **Gestion des erreurs** : Affichage d'un message d'erreur en cas de problème avec l'API.
- **Interface utilisateur** : Design simple avec un loader pendant le chargement, et une présentation claire des informations.
- **Icônes météo** : Utilisation d'icônes SVG stockées dans le dossier `public/icons/` pour représenter les conditions météorologiques.

### Technologies utilisées

- **React** : Bibliothèque JavaScript pour la construction de l'interface utilisateur.
- **Vite** : Outil de build rapide pour les projets front-end.
- **API AirVisual** : Service externe pour obtenir les données météorologiques.
- **CSS** : Styles personnalisés pour l'apparence de l'application.

## Installation et configuration

### Prérequis

- Node.js (version 16 ou supérieure)
- npm ou yarn pour la gestion des dépendances

### Étapes d'installation

1. Clonez le dépôt ou téléchargez les fichiers du projet.
2. Ouvrez un terminal dans le répertoire racine du projet.
3. Installez les dépendances :
   ```bash
   npm install
   ```
